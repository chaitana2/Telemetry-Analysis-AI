# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-25

### Added
- Core `DataLoader` for parsing and preprocessing Toyota telemetry CSVs.
- AI module containing `LapTimePredictor` (LSTM), `AnomalyDetector` (Isolation Forest), and `RaceCoach`.
- Basic PyQt6 `MainWindow` with CSV import functionality and data table view.
- Unit tests for data loading and AI models.
- Initial project documentation (README, CONTRIBUTING, CHANGELOG).
